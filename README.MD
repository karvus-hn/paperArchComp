# # Сравнение нейросетевых архитектур с памятью для локальной навигации мобильного робота в динамических средах с использованием глубокого обучения с подкреплением

Структура модульная и устроена следующим образом

```text
my_project/
│
├── env/
│   └── env2D.py            # Процедурная среда симуляции робота Simple2DEnv
│
├── models/
│   ├── __init__.py         # Словари-регистраторы моделей
│   ├── FrameStack.py       # Архитектура 1: Подход склейки кадров
│   ├── ConvLSTM.py         # Архитектура 2: Сверточно-рекуррентная сеть (conv-lstm)
│   └── AttentionModel.py   # Архитектура 3: Сеть с механизмом внимания (att)
│
├── agent.py                # Модуль с DDPG-агентом , буфером опыта и шум
├── main.py                 # Точка входа: запуск циклов обучения и тестирования
├── requirements.txt        # Список зависимостей фиксированных версий
└── README.md               # Справка-помощь
```

### 1. Клонирование репозитория и установка зависимостей
Рекомендуется использовать виртуальное окружение Python 3.11+:

```bash
git clone https://github.com/karvus-hn/paperArchComp.git
cd paperArchComp
pip install -r requirements.txt
```

### 2. Запуск обучения агента
Для запуска процесса обучения выполните скрипт `main.py`. Выбор одной из трех исследуемых архитектур осуществляется через аргумент `--arch` (доступны параметры строго из списка `ARCH_TYPES = ["fStack", "conv-lstm", "att"]`):

```bash
# Запуск склейки кадров (Frame Stacking)
python main.py --arch fStack

# Запуск сверточно-рекуррентной модели (Conv-LSTM)
python main.py --arch conv-lstm

# Запуск модели с механизмом внимания (Self-Attention)
python main.py --arch att
```
